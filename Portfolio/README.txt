Karthik — Personal Portfolio Website
=====================================

A personal portfolio website by Karthik, built using HTML, CSS, and JavaScript
as part of the web development assignment.


ASSIGNMENT REQUIREMENTS COVERED
--------------------------------
- Homepage with name, profile picture & introduction        [YES]
- Skills section with cards and tags                        [YES]
- Contact section with form (name, email, message)          [YES]
- Responsive design with mobile support                     [YES]
- Clean UI with consistent color scheme                     [YES]
- Responsive navbar (visible on desktop)                    [YES]
- Hamburger toggle menu for mobile                          [YES]
- Dark / Light mode toggle                                  [YES]


FILE STRUCTURE
--------------
Karthik/portfolio/
  index.html   ->  Main HTML, all sections and scripts
  style.css    ->  Stylesheet, theme variables, layout, responsive
  me.png       ->  Profile photo


SECTIONS
--------
- Home    : Profile photo, name, role, and short bio
- Skills  : Cards with tags for Languages (Python, C, C++)
            and Frontend (HTML, CSS, JavaScript)
- Contact : Form with name, email, and message fields


TECH STACK
----------
- HTML5
- CSS3 (Flexbox, Grid, CSS Variables)
- Vanilla JavaScript


FEATURES
--------
- Dark / Light theme toggle
- Hamburger menu for mobile screens
- Smooth scroll navigation
- Responsive layout for mobile and desktop
- Contact form with submit animation
